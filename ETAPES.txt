1: Installer la bibliothèque Swagger dans votre projet Laravel en utilisant Composer :

    composer require darkaonline/l5-swagger

2: Publier les fichiers de configuration de Swagger en utilisant la commande artisan :

    php artisan vendor:publish --provider "L5Swagger\L5SwaggerServiceProvider"


3: On met des annotations dans les fichiers de contrôleurs pour décrire les routes et les méthodes de contrôleurs. Par exemple dans le controlleur de client:
pour créer un client 
        /**
     * @OA\Post(
     *     path="/api/clients",
     *     tags={"Client"},
     *     summary="Créer un nouveau client",
     *     description="Créer un nouveau client",
     *     @OA\RequestBody(
     *         description="Client object",
     *         required=true,
     *         @OA\JsonContent(ref="#/components/schemas/Client")
     *     ),
     *     @OA\Response(
     *         response=200,
     *         description="Successful operation",
     *         @OA\JsonContent(ref="#/components/schemas/Client")
     *     ),
     *     @OA\Response(
     *         response=405,
     *         description="Invalid input"
     *     )
     * )
     */

3 : générer la documentation Swagger. Cela va analyser votre code source et générer un fichier JSON Swagger dans le dossier public/docs/.

    php artisan l5-swagger:generate



On peut accéder au fichier JSON Swagger à l'adresse suivante : http://localhost:8000/docs

On peut accéder à la documentation Swagger à l'adresse suivante : http://localhost:8000/api/documentation
